# 关于站点

 - 基于 HEXO 
 - 使用 Fluid 主题
 - 部署在 Vercel
 - 储存库位于 Github

# 建站原因

2022年苏州516开学事件发生。

Aozaky 建了一个叫做[save516.ICU](http://save516.icu)(目前是DNS污染状态，网站下线)

我们找到了它的[Github仓库](https://github.com/3261596378/Save516.ICU)

页面被部署在了[这儿](https://save516.vercel.app/blogs/origin/)

为了热度不散，我们创建了这个网站。
